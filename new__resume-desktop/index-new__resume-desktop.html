Видимость резюме

Укажите видимость резюме

Если вы активно ищите работу, то этот вариант самый подходящий. Все компании, которые зарегистрированы на AT-WORK, смогут увидеть Ваше резюме. Это повышает шанс найти подходящую для себя вакансию

Ваше резюме смогут просматривать компании, которые вы выберете в списке. Это дает Вам возможность откликаться на все вакансии сайта AT-WORK, но компании, которым вы ограничили просмотр своего резюме, не будут иметь к нему доступ через поиск по базе резюме

Ваше резюме будет доступно для просмотра всем компаниям, которые зарегистрированы на AT-WORK, за исключением тех, которые вы отметите в специальном списке. Таким резюме вы сможете откликаться на все вакансии сайта, однако те компании, которым вы запретили просматривать свое резюме, не будут иметь к нему доступ через поиск по базе данных

Ваше резюме не смогут увидеть компании на сайте AT-WORK по базе данных. Это возможно будет сделать только по прямой ссылке.

Ваше резюме будет недоступно для просмотра всем работодателям, а также не будет выводиться в результатах поиска по базе данных. Вы сможете откликаться таким резюме на заинтересовавшие вас вакансии на AT-WORK

В анонимном резюме можно скрыть персональные данные, фото, контактную информацию или места работы. История откликов и все переписки с работодателем, где уже содержится ваше имя, сохранятся
